Генерация паспортов НКУ

Скрипт создаёт отдельные Word-паспорта из Word-шаблона и данных реестра. Источник данных выбирается при запуске: онлайн Google Таблица или локальный Excel.

Скачать DOCX-инструкциюСкачать файлы запуска

Что делает скрипт

Берёт строки реестра, заменяет метки в passport_template_v2.docx и сохраняет готовые документы в папку Gotovye_pasporta.

Google SheetsExcelWord DOCX
Папка

Что должно лежать рядом

Минимальная структура
PassportGenerator/
├─ generate_passports_v7_Onln.py
├─ passport_template_v2.docx
├─ google_service_account.json                 ← для онлайн-режима
├─ registry.xlsx                                ← для локального режима
├─ run_generate_passports_interactive_Onln_macOS_auto_venv.command
└─ run_generate_passports_interactive_Onln_Windows.bat
Важно: для локального режима Excel-файл должен быть закрыт. Для онлайн-режима таблица должна быть расшарена на service account с правами редактора.
Быстрый запуск

Запуск двойным кликом

Рекомендуемый способ для обычных пользователей — использовать готовый файл запуска под свою операционную систему.

macOS

Запуск через файл run_generate_passports_interactive_Onln_macOS_auto_venv.command.

Windows

Запуск через файл run_generate_passports_interactive_Onln_Windows.bat.

Команды для macOS после скачивания .command

Один раз выдать права на запуск:

Права на запуск
chmod +x /путь/к/run_generate_passports_interactive_Onln_macOS_auto_venv.command

Если macOS заблокировала файл после скачивания:

Снять quarantine-метку
xattr -d com.apple.quarantine /путь/к/run_generate_passports_interactive_Onln_macOS_auto_venv.command
Удобный способ: в Terminal напишите chmod +x , затем перетащите файл .command в окно Terminal и нажмите Enter.
Ручной запуск

Команды запуска без интерактивного файла

Онлайн-режим: передаётся ссылка на Google Таблицу с нужным gid.

macOS
python3 generate_passports_v7_Onln.py "https://docs.google.com/spreadsheets/d/ID_ТАБЛИЦЫ/edit?gid=GID_ЛИСТА#gid=GID_ЛИСТА"
Windows
py -3 generate_passports_v7_Onln.py "https://docs.google.com/spreadsheets/d/ID_ТАБЛИЦЫ/edit?gid=GID_ЛИСТА#gid=GID_ЛИСТА"

Локальный режим: скрипт запускается без ссылки и ищет Excel рядом с собой, как в старом режиме.

macOS
python3 generate_passports_v7_Onln.py
Windows
py -3 generate_passports_v7_Onln.py
