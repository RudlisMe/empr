# EMPR automation landing

Статический лендинг для рабочих инструкций EMPR:
- генерация паспортов НКУ;
- заполнение реестра НКУ;
- работа с Telegram-ботом `@EMPR_smetaBot` для смет, пересчёта `final_result`, контроля проблемных сумм, онлайн-ТКП и admin-команд.

Главная страница: `index.html`.
Страница бота: `bot.html`.
