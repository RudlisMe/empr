fill_registry_nku_Onln_v9.py

Заполнение реестра НКУ

Скрипт помогает добавлять позиции в реестр: формирует тип изделия, присваивает номера, переносит ток и напряжение, сохраняет данные в Google Таблицу или локальный Excel.

Скачать DOCX-инструкциюСкачать файлы запуска

Текущие возможности

Онлайн-режим работает по ссылке на Google Таблицу. Локальный режим можно запускать без указания Excel-файла: скрипт сам найдёт один .xlsx рядом с собой.

v9 OnlnGoogle SheetsExcel
Папка

Что должно лежать рядом

Минимальная структура
RegistryNKU/
├─ fill_registry_nku_Onln_v9.py
├─ google_service_account.json                 ← для онлайн-режима
├─ registry.xlsx                                ← для локального режима
├─ run_fill_registry_nku_interactive_Onln_macOS.command
└─ run_fill_registry_nku_interactive_Onln_Windows.bat
Важно: JSON-ключ никому не отправлять и не выкладывать. Для каждой новой Google Таблицы нужно выдать service account права редактора.
Быстрый запуск

Запуск двойным кликом

Файл запуска создаёт .venv, обновляет pip, проверяет зависимости и спрашивает режим работы.

macOS

Используйте run_fill_registry_nku_interactive_Onln_macOS.command.

Windows

Используйте run_fill_registry_nku_interactive_Onln_Windows.bat.

Команды для macOS после скачивания .command

Один раз выдать права на запуск:

Права на запуск
chmod +x /путь/к/run_fill_registry_nku_interactive_Onln_macOS.command

Если macOS заблокировала файл после скачивания:

Снять quarantine-метку
xattr -d com.apple.quarantine /путь/к/run_fill_registry_nku_interactive_Onln_macOS.command
Удобный способ: в Terminal напишите chmod +x , перетащите файл .command в окно Terminal и нажмите Enter.
Ручной запуск

Команды запуска без интерактивного файла

Онлайн-режим: передаётся ссылка на Google Таблицу и файл учётных данных service account.

macOS
python3 fill_registry_nku_Onln_v9.py "https://docs.google.com/spreadsheets/d/ID_ТАБЛИЦЫ/edit?gid=GID_ЛИСТА#gid=GID_ЛИСТА" --credentials google_service_account.json
Windows
py -3 fill_registry_nku_Onln_v9.py "https://docs.google.com/spreadsheets/d/ID_ТАБЛИЦЫ/edit?gid=GID_ЛИСТА#gid=GID_ЛИСТА" --credentials google_service_account.json

Локальный режим: если рядом лежит один .xlsx, путь можно не указывать.

macOS
python3 fill_registry_nku_Onln_v9.py
Windows
py -3 fill_registry_nku_Onln_v9.py
Ввод данных

Как проходит заполнение

1. Выбор параметров

Скрипт пошагово спрашивает конструктивное исполнение, класс, группу, серию, ток, напряжение и климатическое исполнение.

2. Формирование типа

После выбора параметров формируется тип изделия НКУ.

3. Запись строки

Вводятся номер заказа, IP, размер, вес. После полной строки данные сохраняются.

Подсказка: если в вопросе есть значение в квадратных скобках, можно нажать Enter — будет повторено предложенное значение.
