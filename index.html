Установка Python и библиотек

Действия выполняются один раз на компьютере, где будут генерироваться паспорта.

Windows

  1. Скачайте Python с официального сайта python.org/downloads.
  2. Запустите установщик и обязательно поставьте галочку Add python.exe to PATH.
  3. Откройте PowerShell или Command Prompt и проверьте установку.
Проверить Python
py -3 --version

Установите библиотеки для работы с Word и Excel:

Установить библиотеки
py -3 -m pip install python-docx openpyxl

macOS

  1. Откройте Terminal: Cmd + Space → Terminal → Enter.
  2. Проверьте, установлен ли Python.
Проверить Python
python3 --version

Если Python не найден, скачайте установщик с python.org/downloads/macos. Затем установите библиотеки:

Установить библиотеки
python3 -m pip install python-docx openpyxl
Workflow

Как пользоваться генератором

Скрипт рассчитан на папку с тремя основными файлами.

Структура папки
PassportGenerator/
├─ generate_passports.py
├─ passport_template_v2.docx
├─ registry.xlsx
└─ Gotovye_pasporta/    ← создаётся автоматически
Важно: перед запуском закройте registry.xlsx в Excel, чтобы Python мог прочитать файл без блокировки.

Запуск на Windows

Откройте папку с файлами, нажмите правой кнопкой в пустом месте → Open in Terminal, затем выполните:

Запустить генератор
py -3 generate_passports.py

Запуск на macOS

Откройте Terminal, перейдите в папку и запустите скрипт:

Перейти в папку
cd ~/Desktop/test
Запустить генератор
python3 generate_passports.py

Выбор строки, с которой начинать

Так как Excel постоянно пополняется, генератор не обязан каждый раз создавать паспорта по всей таблице.

При запуске скрипт спросит:

Start Excel row for generation [press Enter for 3]:

Нажать Enter

Генерация начнётся с первой строки с данными.

Ввести 20

Генерация начнётся с 20-й строки Excel и продолжится до конца таблицы.

Логика работы скрипта

Скрипт читает значения из колонок Excel и заменяет метки в Word-шаблоне.

Колонка ExcelМетка WordПример результата
Номер заказа{{ORDER_NO}}1617
Тип изделия{{PRODUCT_TYPE}}Щит ОЩО НКУ‑Ш8101‑2260 У3
Зав.№{{FACTORY_NO}}2002
IP{{IP}}IP31 или 31 — зависит от шаблона
Ток, А{{CURRENT_DISPLAY}}20А или Н/П, если значение 0
Напряжение, В{{VOLTAGE_DISPLAY}}380В или Н/П, если значение 0
Размер{{SIZE}}260x340x120
Вес{{WEIGHT_DISPLAY}}11кг или ___, если ячейка пустая
Правило: если ток или напряжение равны 0, в паспорт подставляется Н/П. Если вес пустой, подставляется ___.

FAQ

Частые ошибки

Ошибка: No module named docx

Не установлена библиотека python-docx. Выполните команду установки библиотек для вашей системы.

Ошибка: Word template was not found

Проверьте, что рядом со скриптом лежит файл passport_template_v2.docx. Название должно совпадать полностью.

Скрипт создал старые паспорта повторно

При запуске укажите номер строки, с которой нужно начать генерацию новых изделий.

В паспорте остались метки {{...}}

Значит в шаблоне есть метка, которую скрипт не знает, или в Excel нет соответствующей колонки. Проверьте таблицу соответствия выше.
