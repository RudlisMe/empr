Telegram-бот для реестра и паспортов НКУ

Бот работает на VPS как системная служба. После команды /start он просит ссылку на Google Таблицу, затем предлагает: заполнить реестр или сгенерировать паспорта по уже внесённым строкам.

Что умеет бот

Заполняет онлайн-реестр, генерирует DOCX-паспорта, отправляет файлы в Telegram и возвращает пользователя в меню действий.

Сценарий

Как пользователь работает с ботом

1. Отправить /start

Бот просит ссылку на Google Таблицу. Таблица должна быть доступна service account с правами редактора.

2. Выбрать действие

Доступны два режима: Заполнять реестр и Сгенерировать паспорта.

3. Получить результат

При генерации бот отправляет готовые DOCX-файлы или ZIP-архив прямо в окно Telegram.

Важно: если после перезапуска VPS старые кнопки в Telegram не реагируют, отправьте /start. Старые кнопки могут остаться от предыдущей сессии.
Что нужно заранее

Минимальный набор для запуска

VPS

Подойдёт обычный Ubuntu VPS с доступом по SSH. Бот запускается через systemd и продолжает работать после закрытия SSH.

Telegram token

Создаётся через @BotFather. Token не хранится в GitHub, а вводится при установке.

Google JSON

Нужен google_service_account.json. Его нельзя выкладывать в репозиторий или отправлять посторонним.

Новый VPS

Разворачивание одной командой

Команда скачивает install-скрипт из GitHub, устанавливает зависимости, создаёт виртуальное окружение, настраивает .env, принимает JSON-ключ и запускает службу.

Команда для чистого Ubuntu VPS
apt update && apt install -y curl git && curl -fsSL https://raw.githubusercontent.com/RudlisMe/nku-registry-bot/main/install_on_new_vps.sh -o /tmp/install_nku_bot.sh && REPO_URL=https://github.com/RudlisMe/nku-registry-bot.git bash /tmp/install_nku_bot.sh
Во время установки скрипт спросит: Telegram BOT_TOKEN, Telegram ID разрешённых пользователей и содержимое google_service_account.json. После вставки JSON нужно нажать Enter, затем Ctrl+D.
Понятно по шагам

Что делает команда установки

  1. Устанавливает системные пакеты: Python, venv, pip, git, unzip, nano.
  2. Клонирует репозиторий RudlisMe/nku-registry-bot в папку /opt/nku_bot.
  3. Создаёт виртуальное окружение /opt/nku_bot/.venv.
  4. Устанавливает зависимости из requirements.txt.
  5. Создаёт .env и принимает секреты вручную.
  6. Создаёт службу nku-bot и включает автозапуск.
